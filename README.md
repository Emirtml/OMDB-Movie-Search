# Telco Customer Churn Prediction

This project focuses on predicting customer churn for a telecommunications company. I used a dataset containing information about 7000+ customers to build a machine learning model.

## Steps I followed:
1. **Data Cleaning:** Fixed the data type of 'TotalCharges' and handled missing values.
2. **Preprocessing:** Converted categorical text data into numbers using One-Hot Encoding so the model can understand it.
3. **Modeling:** Used the **RandomForestClassifier** to train the data.
4. **Evaluation:** Checked the performance using accuracy score and classification report.

## Libraries Used:
* Pandas & NumPy
* Scikit-Learn
* Matplotlib/Seaborn

## How to Run:
Just open the `churn_modeli.ipynb` file in a Jupyter environment and run all cells.
